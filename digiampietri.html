Gerenciamento de Processos de Negócio / Workflows Científicos
Objetivo: A proliferação de dados e ferramentas utilizados em experimentos científicos ou industriais originou diversos desafios: como entender e organizar esses recursos, como compartilhar e reusar experimentos bem sucedidos, e como prover interoperabilidade entre dados e ferramentas de diferentes locais e utilizados por usuários com perfis distintos. Análise de Redes Sociais e Cientometria

Neste grupo de pesquisa desenvolvemos diferentes algoritmos para a análise de redes sociais, com enfoque em algoritmos para a predição de relacionamentos, identificação de especialistas, descoberta de tendências, recomendações em redes sociais e análise de redes sociais acadêmicas. Particularmente estamos interessados na: (i) identificação, (ii) avaliação de medidas, e (iii) descoberta de conhecimento relacionadas à produção técnica-científica, seja esta representada por produção bibliográfica, formação de profissionais e colaboração acadêmica..

Gerenciamento e Composição Automática de Serviços Web Semânticos
Descrição: Serviços Web representam uma tecnologia extremamente relevante para prover interoperabilidade. Um importante passo para o desenvolvimento de aplicações complexas baseadas em Serviços Web é a habilidade de selecionar e integrar serviços heterogêneos de oriundos de diferentes fontes. Além disso, sempre que não for possível encontrar um único serviço capaz de desempenhar uma certa tarefa, seria importante a existência de meios (automáticos e semiautomáticos) para compor serviços básicos de forma que, combinados, estes executem a tarefa desejada. Este projeto de pesquisa visa estender a pesquisa realizada durante o doutorado do proponente realizado no Instituto de Computação da UNICAMP e intitulado ?Gerenciamento de Workflows Científicos em Bioinformática?. Além da generalização da aplicação de Serviços Web para todas as áreas de conhecimento, este projeto visa explorar o conceito de Serviços Web Semânticos de forma a permitir que sistemas de software, baseados em Planejamento em Inteligência Artificial, possam compor serviços básicos para desempenharem tarefas mais complexas. Adicionalmente, este projeto irá combinar os recursos oferecidos pelos modelos ontológicos para anotação de Serviços Web Semânticos (WSMO, WSDL-S e OWL-S) com os padrões de workflows (?Workflow Patterns?) a fim de possibilitar o desenvolvimento de uma infraestrutura robusta e extensível para o gerenciamento e composição de Serviços Web Semânticos..

2009 - 2011
Uma infraestrutura para otimização de bancos de dados e controle de carga de trabalho
Descrição: Atualmente, otimizadores de bancos de dados utilizam índices e visões materializadas (VMs) para produzir planos de execução de consultas. Enquanto índices e VMs podem acelerar a execução de consultas, existe um custo para armazená-los e gerenciá-los. Este projeto pretende especificar e implementar uma infraestrutura capaz de recomendar um conjunto de índices e VMs com o propósito de otimizar um banco de dados, dadas uma carga de trabalho e a restrição de espaço disponível em disco. A solução almejada será genérica, podendo ser usada em qualquer sistema de gerenciamento de banco de dados relacional que utilize VMs na seleção de planos de execução de consultas. Enquanto o objetivo principal está relacionado a bancos de dados frequentemente atualizados, a solução almejada também poderá ser utilizada em bancos de dados estáticos (apenas de leitura) e na estimativa do comportamento de novos bancos de dados (com cargas de trabalho pré-definidas)..

Estudo e desenvolvimento de técnicas de processamento de imagens e vídeos digitais
Descrição: O aumento da oferta de câmeras fotográficas e filmadoras digitais (cada dia melhores e mais baratas), bem como do acoplamento de câmeras digitais a microscópios, equipamentos de raio X e outros equipamentos utilizados em diagnósticos e pesquisa criaram novos desafios para o processamento, análise e organização das imagens e vídeos produzidos. O ser humano apresenta uma grande capacidade de olhar para uma imagem e reconhecer objetos ou temas, o que permite uma categorização eficiente de um conjunto de imagens. Mesmo assim, essa categorização manual é sujeita a falhas ocasionadas, por exemplo, pelo cansaço ou mesmo pelo caráter subjetivo de interpretação/entendimento de uma dada imagem. Desta forma são necessários mecanismos que automatizem processos de classificação e interpretação de imagens e vídeos.O objetivo geral deste projeto é desenvolver um conjunto de funções (ou biblioteca) para o processamento de imagens e vídeos. Esse conjunto de funções possuirá funções básicas para o processamento de imagens; funções para a recuperação de imagens por contéudo; e também funções específicas para o processamento de imagens relacionadas a LIBRAS (Língua Brasileira de Sinais) e para a análise morfológica de imagens relacionadas a biomedicina..

Em particular, este projeta aborda soluções relacionadas a Bancos de Dados (otimização de consultas e integração de dados) e Inteligência Artificial aplicados detecção de fraudes e auxilio no preenchimento correto de formulários..
